API Documentation

1. Upload Identity Card Image

Endpoint: POST /api/process-upload

Description: Upload an image of the identity card. The system will process the image to detect QR codes.

Request:

            
                curl -X POST -F "file=@path/to/your/image.png" http://localhost:8080/api/process-upload

Response:

            
                {
                    "status": "success",
                    "message": "Image processed successfully",
                    "update_required": true,
                    "existing_data": {
                        "id_identity_card": "123456",
                        "id_passport": "AB123456",
                        "fullname": "John Doe",
                        "birth_date": "2000-01-01",
                        "gender": "Male",
                        "address": "123 Main St",
                        "issue_date": "2021-01-01"
                    }
                }

2. Update Identity Card

Endpoint: POST /api/update-card

Description: Update an existing identity card with new data.

Request:

            
                curl -X POST -H "Content-Type: application/json" -d '{
                    "id_passport": "AB123456",
                    "id_identity_card": "654321",
                    "fullname": "Jane Doe",
                    "birth_date": "1990-01-01",
                    "gender": "Female",
                    "address": "456 Elm St",
                    "issue_date": "2022-01-01"
                }' http://localhost:8080/api/update-card

Response:

            
                {
                    "status": "success",
                    "message": "Identity card updated successfully."
                }

3. Add or Update Identity Card

Endpoint: POST /api/add-card

Description: Add a new identity card or update an existing one.

Request:

            
                curl -X POST -H "Content-Type: application/json" -d '{
                    "id_identity_card": "654321",
                    "id_passport": "AB123456",
                    "fullname": "Jane Doe",
                    "birth_date": "1990-01-01",
                    "gender": "Female",
                    "address": "456 Elm St",
                    "issue_date": "2022-01-01"
                }' http://localhost:8080/api/add-card

Response:

            
                {
                    "status": "success",
                    "message": "Identity card added or updated successfully."
                }

4. Delete Identity Card

Endpoint: DELETE /api/delete-card/{idPassport}

Description: Delete an identity card based on the passport ID.

Request:

            
                curl -X DELETE http://localhost:8080/api/delete-card/AB123456

Response:

            
                {
                    "status": "success",
                    "message": "Identity card deleted successfully."
                }

5. Get All Identity Cards

Endpoint: GET /api/cards

Description: Retrieve a list of all identity cards.

Request:

            
                curl -X GET http://localhost:8080/api/cards

Response:

            
                [
                    {
                        "id_identity_card": "654321",
                        "id_passport": "AB123456",
                        "fullname": "Jane Doe",
                        "birth_date": "1990-01-01",
                        "gender": "Female",
                        "address": "456 Elm St",
                        "issue_date": "2022-01-01"
                    }
                ]

6. Get Card Image

Endpoint: GET /api/card-image/{idPassport}

Description: Retrieve the image associated with the identity card.

Request:

            
                curl -X GET http://localhost:8080/api/card-image/AB123456

Response: The image file of the identity card.
